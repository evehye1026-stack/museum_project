## 1. 프로젝트 개요

국립중앙박물관 관람객에게 전시 유물을 주제로 한 **추천 관람 코스**를 제공하는 서비스.
사용자는 코스별 소요 시간, 경유 유물, 관람 순서(Step)를 확인하며 박물관을 효율적으로 둘러볼 수 있다.

- **핵심 가치**: 사색(思索), 여유, 전통과 현대의 조화
- **타깃**: 박물관 방문객(내외국인), 전시 동선을 미리 계획하고 싶은 사용자
- **플랫폼**: 반응형 웹(Desktop) + 모바일 웹/앱

## 2. 디자인 컨셉

### 브랜드명 "처마"
한옥 지붕의 **처마선(曲線)** 을 그래픽 모티프로 사용. 로고는 처마 끝의 곡선을 라인아트로 단순화한 형태.

### 컨셉 키워드
| 키워드 | 설명 |
|---|---|
| 전통 × 현대 | 전통 건축 요소(처마) + 미니멀한 웹 UI |
| 사색의 길 | 유물을 감상하며 걷는 여정을 "길"로 은유 |
| 정적인 무게감 | 화려하지 않은 네이비 톤으로 유물의 위엄을 해치지 않음 |

### 무드
차분하고 지적인 느낌. 박물관이라는 공간의 무게감을 존중하면서도, 디지털 서비스로서 답답하지 않은 여백과 곡선을 사용.

## 3. 타이포그래피

참고 이미지의 "한칸인의 명조 & 고딕" 문구는 **명조체 + 고딕체 페어링**을 의미.

| 용도 | 서체 계열 | 사용처 |
|---|---|---|
| 헤드라인 / 코스 타이틀 | 명조(Serif, 예: 본명조, 노토 세리프 KR) | "반가사유상과 함께하는 사색의 길" 등 전시 코스명 |
| 본문 / UI 텍스트 | 고딕(Sans, 예: 프리텐다드, 노토 산스 KR) | 설명문, 버튼, 내비게이션, 라벨 |

- 코스명·유물명 등 **감성적 콘텐츠**는 명조로 무게감을 주고,
- 메뉴·버튼·안내문 등 **기능적 UI**는 고딕으로 가독성을 확보하는 원칙.

**타입 스케일(제안)**
- H1 (페이지 타이틀): 32–40px, 명조, Bold
- H2 (섹션/코스 타이틀): 24–28px, 명조, Bold
- Body: 15–16px, 고딕, Regular
- Caption/Label (step, 시간 표시): 12–13px, 고딕, Medium

## 6. 주요 컴포넌트

### 6.1 코스 카드 (Course Card)
- 배경: `#1B2A5E → #5C73A5` 대각선 그라데이션
- 오버레이: 저채도 원형/구름형 그래픽(투명도 10–20%)로 배경에 깊이감
- 콘텐츠: 박물관명(소, 고딕) → 코스 타이틀(대, 명조) → 대표 유물 이미지(우측 정렬, 라이팅 강조)
- 카드 하단: 사용된 컬러 태그(점 + HEX) — 코스별 테마 컬러를 보여주는 장식 요소로 활용 가능

### 6.2 Step 프로그레스 인디케이터
- 가로형 타임라인, 점선으로 노드 연결
- 노드마다: Step 번호 배지 + 아이콘 + 소요시간(예: 10min/30min) + 컬러 코드
- 완료/진행/예정 상태를 컬러 및 채움 정도로 구분 (예: 진행 전 = 연한 톤, 현재 = `#0078FF` 강조)

### 6.3 헤더 내비게이션
- 흰 배경 + 하단 얇은 보더(그림자 없이 플랫)
- 로고는 처마 라인아트, 스크롤 시에도 고정(sticky header)

### 6.4 추천 코스 리스트
- 카드형 그리드(데스크톱 3열 / 모바일 1열 캐러셀)
- 각 카드: 썸네일 + 코스명(명조) + 예상 시간 + 난이도/거리 뱃지

## 7. 그래픽 모티프 & 이미지 스타일

- **라인아트 아이콘**: 한옥 처마·기와 곡선을 단순화한 1px 라인 드로잉 → 로고, 섹션 구분 장식에 활용
- **추상 원형/구름 패턴**: 카드 배경의 저채도 오버레이로 반복 사용 (전통 문양 느낌 + 미니멀함 유지)
- **유물 사진**: 어두운 배경 위에서 스포트라이트 받듯 밝게 보정 — 유물이 카드의 시각적 초점이 되도록 우측 정렬 배치
- **아이콘 스타일**: 얇은 선(outline) 아이콘, 컬러는 본문과 동일한 네이비 계열 사용
